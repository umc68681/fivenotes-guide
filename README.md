## FiveNotes 3 是什么？为什么只有五个笔记

用过越方便的笔记工具的人越容易犯一个毛病：笔记越记越多，找东西反而更难了。

五个页面，就是刻意设置的上限。不能无限堆积，倒逼你保持整洁。官方的说法是「五个点，再也不多了（Five dots. No more.）」，听起来像禅意广告语，但真的用起来，逻辑是通的——桌上堆了一百张便利贴你不会用，但备好五张空白的，你会随手就写。

五个笔记页通过窗口底部的彩色圆点切换，快捷键 `Cmd+Shift+[` 和 `Cmd+Shift+]` 在页面间跳转，或者直接按 `Cmd+数字` 跳到对应页。窗口可以固定置顶、拖拽到屏幕任意位置，也支持跟随鼠标光标显示——不管你在什么 app 里工作，抬起手腕记一句话，降回去继续干活。

## FiveNotes 3 有哪些核心功能

### 菜单栏秒速呼出

FiveNotes 常驻系统菜单栏，点击图标即开窗。你可以设置自定义全局快捷键，让它在任何情境下弹出。笔记内容自动保存，不会因为关窗而丢失。

### 悬浮置顶，不打断当前工作

点击窗口右上角的图钉按钮，FiveNotes 就钉在屏幕最前方，盖在其他所有窗口上面——包括全屏应用。你一边看参考文档，一边在 FiveNotes 里记关键词，两件事互不干扰。

### Markdown 支持

支持基础 Markdown 格式：标题用 `#`，加粗用 `**文字**`，列表用 `-`，足够日常使用。配合不同主题，渲染出来还挺好看。

### iCloud 跨设备同步

笔记内容经由 iCloud 同步，多台 Mac 之间实时更新。有 iOS / iPadOS 版的 FiveNotes Mobile（单独购买），手机和平板也能接着用。

### 自动备份

定期自动备份笔记历史，写了一堆突然手滑删掉也能找回来。

### 多彩主题

内置多种配色方案，支持在线下载社区主题，支持系统深色/浅色模式自动切换。你还可以下载免费的 ThemeEdit 应用自制主题。

### 自动化集成

提供 Apple Script API、URL API 和 Apple 快捷指令（Shortcuts）支持，可以和其他工作流工具打通。开发者可以通过脚本往 FiveNotes 写入内容，或从中读取记录。

## FiveNotes 3 适合哪些场景？谁用它最顺手

**创作者和写作者**：灵感弹出来那一秒最宝贵。不用开文档、不用新建文件、不用命名——直接写进去，回头整理。

**程序员和开发者**：调试时临时记一个报错信息、一个参数值，或者跨应用搬运几段代码片段。配合 URL API 还能从脚本里直接写入笔记。

**学生和研究者**：听课、读文献时遇到要记的东西，不用切出去打开大型应用。笔记页也可以用来存常查的公式、定义、缩写表，置顶显示当备忘单用。

**多任务工作者**：会议中临时记行动项，不想打断会议节奏；同时处理多个项目，每个笔记页对应一个上下文。

不适合的场景说清楚：如果你需要长篇笔记、多层级文件夹、协作分享、图片附件，FiveNotes 不是合适的工具。它的定位就是速记暂存，不是知识管理系统。

## FiveNotes 3 和同类工具怎么比

说实话，菜单栏便签这个品类并不冷门。macOS 自带便笺、Tot、Notchmeister、Paste Notes……各有各的思路。

Tot 是目前和 FiveNotes 相互对比最多的一款。两款在外观上有相似之处，但 FiveNotes 官网明确指出：FiveNotes 出现更早（前身 Quick 在 Tot 之前），并且在 FiveNotes 3 中做了大幅升级。实际区别在于：FiveNotes 对自动化集成支持更好（AppleScript / URL API），主题生态丰富，还有 iOS/Watch 扩展；Tot 的界面设计更保守，功能更克制。

macOS 自带的「便笺」（Stickies）虽然免费，但没有全局快捷键、没有 iCloud 同步、没有主题，界面也停留在很多年前。V2EX 上有用户提到试了一圈之后选择 FiveNotes，原因是「系统便笺每次删掉都要再点一次确认，还有格式问题」——这种小摩擦，日积月累真的烦。

## FiveNotes 3 安全吗？笔记内容会被上传吗

根据 Apptorium 的隐私政策：FiveNotes 不收集、不使用、不存储、也无法访问你的笔记内容。你的笔记保存在本地设备上，通过你自己的 iCloud 账号同步，Apptorium 无权查看。应用本身不追踪使用行为，不收集任何分析数据。

如果通过 Mac App Store 购买，Apptorium 不会收到你的个人信息；官网购买由 Paddle 处理支付，仅共享完成交易所需的必要信息。

隐私这一点，相比一些把笔记内容存到开发商服务器的 SaaS 工具，FiveNotes 的本地存储 + 私有 iCloud 路径确实更干净。
